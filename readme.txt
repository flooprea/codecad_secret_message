Codecademy mini-project on Arrays.

Description: Using array methods, you will transform the below array of strings into a secret message!
let secretMessage = ['Learning', 'is', 'not', 'about', 'what', 'you', 'get', 'easily', 'the', 'first', 'time,', 'it', 'is', 'about', 'what', 'you', 'can', 'figure', 'out.', '-2015,', 'Chris', 'Pine,', 'Learn', 'JavaScript'];

Extra features added:
    The user can click the button to be promted with a text box. In this text box they can input the element that the they want to remove from the array.
